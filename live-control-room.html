CyberCrowd Live Control Room

CyberCrowd Live Control Room is the coordination layer for the multi-surface live system. It is where Magic Cursor, phone input, monitors, camera packages, show surfaces, audience interaction, and archive paths can be controlled as one live operating environment.

Control Room Purpose

CyberCrowd is not a single feed and not a one-screen app. The Live Control Room exists to coordinate multiple surfaces at once: phones, cameras, monitors, live rooms, public discourse, show layers, media vessels, and trusted control paths.

Multi-Surface Layout

  • Phone input and mobile camera control
  • Monitor A: Atomic Kricket show or culture surface
  • Monitor B: CyberCrowd control and public discourse surface
  • Monitor C: Reten Records music, video, and stage arts surface
  • Camera package controls
  • Viewer upgrade controls
  • Audience interaction surfaces
  • Archive and proof capture paths

Magic Cursor Coordination

Magic Cursor is the control concept that can move across surfaces, guide attention, select inputs, route media, and help coordinate the live system without forcing everything into one flat screen.

Where We Belong

The Live Control Room supports CyberCrowd’s placement logic. A camera, file, person, signal, surface, record, show, or proof object must be routed to where it belongs. WDIG and Magic Cursor help define, route, and control that placement.

Connection to Cam Packages

Cam Packages define the available cameras and live views. The Live Control Room provides the coordination layer for using those cameras, switching perspectives, managing access, and connecting the event to the broader CyberCrowd system.

Connection to Viewer Upgrades

Viewer Upgrades define what the audience can unlock. The Live Control Room helps operators understand what views, surfaces, or event paths are active and what access levels are available.

Archive and Proof Path

A live event should not vanish into a feed. CyberCrowd can preserve event structure, control decisions, camera configurations, routing choices, access states, and incident notes through the CyberCrowd Archive when needed.

Connection to the CyberCrowd System

The Live Control Room may connect to WDIG routing, Magic Cursor, Cam Packages, Viewer Upgrades, Secure Surface Hardware, Atomic Kricket, Reten Records, public discourse, audience interaction, and platform archive proof.
