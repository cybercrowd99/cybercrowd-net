CyberCrowd Event Access

CyberCrowd Event Access defines how a specific live event, live room, camera package, viewer experience, or creator/client session controls who can enter, what can be viewed, what can be upgraded, and what may be preserved.

Event Access Purpose

CyberCrowd access should be tied to the event context. A viewer, creator, client, operator, or surface may have permission for one event, one room, one camera package, one replay window, or one controlled session without receiving unlimited platform access.

Possible Event Access Types

  • Public event access
  • Private event access
  • Invite-only event access
  • Trial event access
  • Single-camera event access
  • Multi-camera event access
  • Premium angle event access
  • Replay or VOD event access
  • Creator/operator event access
  • Client-controlled event access

Connection to Cam Packages

Cam Packages define the available cameras and angles for an event. Event Access defines who can view those cameras, which angles are included, which angles require upgrades, and what remains operator-only.

Connection to Viewer Upgrades

Viewer Upgrades may unlock additional event views, premium angles, behind-the-scenes access, replays, VOD windows, or event-specific features.

Connection to Trial Access

Trial Access may allow a viewer to test part of an event before upgrading. The event should define what the trial includes, how long it lasts, and what happens when the trial ends.

Creator and Client Event Controls

Creator Control may allow approved users to prepare or manage the event: active cameras, viewer access levels, trial availability, upgrades, replay windows, archive markers, and incident notes.

Event Proof

Event Access decisions may become proof objects when they affect viewer disputes, creator/client trust, access rules, billing questions, camera availability, or platform review.

Connection to the CyberCrowd System

Event Access may connect to Cam Packages, Viewer Upgrades, Trial Access, Access Passes, Creator Control, Live Control Room, Device Trust, WDIG Routing, Magic Cursor, Client Investigation, Platform Archive, adWorm, Atomic Kricket, and Reten Records media events.
