CyberCrowd Creator Control

CyberCrowd Creator Control is the management layer for creators, clients, venues, operators, and approved users running live rooms, cam packages, viewer access, multi-surface events, and archive paths.

Creator Control Purpose

CyberCrowd is not only a place to upload content. It is a live control system where creators and clients may define what is active, what is visible, what can be upgraded, what gets archived, and which surfaces belong inside the event.

Who May Use Creator Control

  • Creators
  • Clients
  • Venue operators
  • Live room operators
  • Show producers
  • Camera operators
  • Approved assistants
  • Future Reten Records media operators
  • Atomic Kricket show operators

Possible Control Areas

  • Start or prepare a live room
  • Select active camera package
  • Manage camera angles
  • Define viewer access levels
  • Control trial or upgrade availability
  • Route media through WDIG
  • Coordinate surfaces through Magic Cursor
  • Mark archive/proof events
  • Manage event notes and incident signals
  • Prepare replay or VOD paths

Connection to Cam Packages

Cam Packages define what cameras and views are available. Creator Control helps determine which package is active, which angles are public, which angles are premium, and which views are reserved for operators or clients.

Connection to Viewer Upgrades

Viewer Upgrades define what the audience can unlock. Creator Control helps operators understand what access options are available during a live event without turning the event into a confusing feed.

Access Discipline

Creator Control must remain permissioned, clear, and auditable. CyberCrowd should define who can control a live room, who can change access, who can route media, and what gets preserved as proof.

Connection to the CyberCrowd System

Creator Control may connect to WDIG Routing, Magic Cursor, Live Control Room, Cam Packages, Viewer Upgrades, Secure Surface Hardware, Archive proof, access passes, Atomic Kricket, and Reten Records media events.
